# Friends-Network-API

* Javascript
* Node.js
* MongoDB
* Mongoose
* Express.js


## Installation

Install the following node packages on command line before you start the app if you would like to use it locally.
  
`npm init`

`npm install`

`npm install express`

`npm install mongoose`

  
## Usage   
  
Run the following commands at the root of your project;

`npm start`
